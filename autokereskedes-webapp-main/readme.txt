Nyisd meg a PyCharm-ot
Másold be a main.py-ba a main.py-t
Csinálj új fájlt (employeeDB.py) -> ide másold be az employeeDB.py-t
Importáld be a flask-et
Aztán írd be, hogy pip install flask_cors
Ha nem akarja, akkor File -> Settings -> Keress rá, hogy 'terminal' -> állítsd át a powerShell-t cmd-re
Ha ezt sem akarja, akkor szedd le a mappákat és a  six.py-t innen
  https://drive.google.com/drive/folders/1z018yyCrZ52zyBsgffSuuDriaAXI4naZ?usp=sharing
És másold be a venv/Lib/site-packages mappába

Aztán másold be fronend fájljait egy "frontend" mappába (ez akár az asztalon is lehet)
Indítsd el az employeeDB.py-t, hogy létrejöjjön az SQLite-os adatbázis
Indítsd el a main.py-t a pyCharmban
Kattints az index.html-re
Ha üres a táblázat, menj át az Add oldalra, töltsd ki néhány mezőt (az első négy mező nem lehet üres, a többi lehet)
Menj vissza az index.html-re, és elvileg meg kell jelennie egy új sornak
Aztán elvileg tudod módosítani az adatokat, vagy törölni a sort
